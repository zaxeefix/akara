# Trust and Safety

## Vendor Trust

- Admin approval before verified listing.
- Business profile completeness.
- Location review.
- Review monitoring.
- Report handling.

## Customer Safety

- Official payment flows.
- Vendor report flow.
- Clear pickup/delivery information.
- Support escalation.

## Reviews

- Prefer reviews tied to completed orders.
- Allow reporting abusive or fake reviews.
- Admin moderation for flagged content.

## Fraud Signals

- Duplicate listings.
- Failed payment spikes.
- Review bursts.
- Suspicious admin access.
- Repeated order cancellations.

## Human Review

High-impact decisions should be reviewed by an admin before suspension, rejection, or restriction.
