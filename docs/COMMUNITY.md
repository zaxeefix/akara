## Community Guidelines

- Keep discussion respectful.
- Do not share secrets or real user data.
- Use issue templates.
- Keep pull requests focused.
- Prefer accessible, mobile-first experiences.
- Keep Nigerian localization and real vendor workflows in mind.

## Good First Issue Ideas

- Add missing translation keys.
- Improve empty states.
- Add tests for API validators.
- Improve docs examples.
- Add accessible labels to new UI controls.
