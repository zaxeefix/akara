# Pilot Vendor Onboarding

## Goal

Bring a small set of trusted vendors into AkaraConnect beta without auto-approving unsafe or incomplete listings.

## Flow

1. Welcome vendor.
2. Create vendor account.
3. Complete business registration.
4. Capture state, LGA, community, street, house number, and GPS coordinates.
5. Upload logo and cover photo when upload provider is ready.
6. Add first menu item.
7. Set opening and closing hours.
8. Select pickup or delivery option.
9. Submit verification.
10. Await admin approval.

## Status Labels

- Draft.
- Submitted.
- Under review.
- Approved.
- Rejected.
- Suspended.

## Admin Review

Admins should check:

- Business name.
- Contact details.
- Vendor location.
- Photos.
- Menu quality.
- Risk flags.
- Duplicate listings.

## Rejection Handling

Provide a clear reason and allow resubmission where appropriate.
