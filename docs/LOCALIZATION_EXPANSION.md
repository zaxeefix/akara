## Expansion Rules

- Do not hard-code user-facing text.
- Add language config and dictionary files.
- Keep keys semantic.
- Test mobile layout with longer translated strings.
- Review translations with fluent speakers.
- Fall back to English for missing keys.

## Suggested Locale Codes

Locale codes should be confirmed before implementation. Avoid inventing codes where standards exist.
