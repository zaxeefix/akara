# Analytics and Growth

## Metrics

- Vendor sales trends.
- Customer growth.
- Food category trends.
- Location-based demand.
- Peak hours.
- Repeat customers.
- Vendor retention.
- Order conversion rate.
- Payment success rate.
- Delivery performance.
- Admin platform health.

## Dashboard Improvements

Vendor insights:

- Best-selling menu items.
- Peak order hours.
- Repeat customer count.
- Low-performing items.
- Estimated profit.

Admin insights:

- Active vendors by state.
- Pending approvals by region.
- Fraud report trends.
- Payment success rate.
- Category demand.

Customer recommendations:

- Nearby popular vendors.
- Frequently ordered foods.
- Recently viewed vendors.

## Data Requirements

- Reliable order events.
- Payment statuses.
- Vendor status changes.
- Search events.
- Review events.
- Location metadata.

## Privacy

Use aggregated analytics by default. Avoid exposing individual customer behavior to vendors unless clearly necessary and consented.
