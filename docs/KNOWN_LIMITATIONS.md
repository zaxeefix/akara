# Known Limitations

## Beta Limitations

- Live provider credentials are not included.
- Upload flows are planned but need real Cloudinary integration.
- Payment flows are Paystack-ready but require live/test provider configuration.
- Auth provider placeholders exist for Google, Phone OTP, and 2FA.
- Maps use safe placeholders until API keys and restrictions are configured.
- Support form is a placeholder until ticket submission is connected.
- Coming Soon modules are intentionally inactive.
- Legal pages require review by qualified counsel before public launch.

## Technical Limitations

- The workspace Git repository is not recognized locally.
- Real E2E testing requires deployed frontend/backend and seeded staging database.
- Webpack cache snapshot warnings may appear during build, but production build completes.
