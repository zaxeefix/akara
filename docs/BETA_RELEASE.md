# Beta Release

## Included in Beta

- Customer marketplace pages.
- Vendor onboarding and dashboard foundation.
- Admin beta operations dashboard.
- Vendor approval workflow structure.
- Search and nearby vendor structure.
- Cart, checkout, order confirmation, and tracking surfaces.
- Review, favorites, notifications, and support surfaces.
- Paystack-ready payment structure.
- Google Maps-ready location structure.
- PWA and SEO foundation.
- Localization foundation for English, Hausa, Yoruba, Igbo, and Tiv.

## Not Yet Included

- Real payment provider live keys.
- Real Google OAuth, Phone OTP, and 2FA provider credentials.
- Real Cloudinary upload endpoints.
- Full WhatsApp Business API bot.
- Real USSD integration.
- Rider production workflow.
- Loan or insurance decisioning.
- Production legal review.

## Feedback Collection
