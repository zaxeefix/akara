# Feature Priority List

## MVP

### Customer

- Email registration and login
- Google login
- Phone verification or phone login foundation
- Customer profile
- Vendor search
- Nearby vendor discovery
- Vendor profile page
- Food category browsing
- Menu viewing
- Cart and order placement
- Order status tracking
- Vendor call action
- Favorite vendors
- Reviews and ratings
- Notifications foundation
- Language switching
- Light mode and dark mode

### Vendor

- Vendor registration
- Complete business profile submission
- Vendor approval status
- Menu management
- Price editing
- Picture uploads through Cloudinary
- Order management
- Opening and closing hours
- Basic delivery or pickup settings
- Customer order history
- Basic sales analytics
- Earnings summary
- Notifications foundation

### Admin

- Admin login
- Role-based admin access
- Vendor approval queue
- Vendor management
- Customer management
- Order visibility
- Basic analytics dashboard
- Review moderation
- Security and audit logs
- Category management

### Platform

- Next.js frontend foundation
- Express.js backend foundation
- TypeScript across frontend and backend
- PostgreSQL on Neon
- Prisma data layer
- Localization files for English, Hausa, Yoruba, Igbo, and Tiv
- Secure auth foundation with NextAuth, JWT, OAuth, and OTP-ready architecture
- Google Maps and geolocation foundation
- Payment abstraction with initial provider integration
- Cloudinary storage integration
- Open-source docs and templates
- GitHub Actions CI foundation
