# User Stories

## Customers

- As a customer, I want to create an account so that I can place orders and save my preferences.
- As a customer, I want to log in with Google so that I can access the platform quickly.
- As a customer, I want to verify my phone number so that vendors can contact me when needed.
- As a customer, I want to search for food or vendors so that I can quickly find what I want.
- As a customer, I want to find vendors near me so that I can buy from convenient locations.
- As a customer, I want to see a vendor's menu, prices, ratings, opening hours, and photos so that I can decide where to order.
- As a customer, I want to open Google Maps navigation so that I can reach a vendor easily.
- As a customer, I want to place an order so that I can reserve or buy food.
- As a customer, I want to pay online or choose cash on delivery so that I can use the payment method that works for me.
- As a customer, I want to track my order so that I know whether it is pending, accepted, preparing, ready, completed, or cancelled.
- As a customer, I want to chat with vendors so that I can clarify availability or pickup details.
- As a customer, I want to call vendors so that urgent order questions can be resolved quickly.
- As a customer, I want to review vendors so that other customers can make informed decisions.
- As a customer, I want to save favorite vendors so that I can reorder faster.
- As a customer, I want notifications so that I do not miss order updates.
- As a customer, I want to use the platform in my preferred language so that the experience feels natural.
- As a customer, I want dark mode and light mode so that I can use the platform comfortably.

## Vendors

- As a vendor, I want to register my business so that customers can discover me online.
- As a vendor, I want to provide detailed location information so that customers can find my shop or stand.
- As a vendor, I want to upload a logo, cover photo, and food pictures so that my storefront looks trustworthy.
- As a vendor, I want to set opening and closing hours so that customers know when I am available.
- As a vendor, I want to manage my menu so that customers see current food options.
- As a vendor, I want to edit prices so that my menu reflects daily cost changes.
- As a vendor, I want to manage order status so that customers know what is happening.
- As a vendor, I want to configure delivery or pickup options so that I can serve customers correctly.
- As a vendor, I want to view customer records so that I can understand repeat customers.
- As a vendor, I want sales analytics so that I can understand business performance.
- As a vendor, I want earnings records so that I can track income.
- As a vendor, I want review notifications so that I can monitor customer feedback.
- As a vendor, I want order notifications so that I can respond quickly.
- As a vendor, I want my profile reviewed by admins so that customers can trust approved vendors.

## Admins

- As an admin, I want to approve vendors so that only valid businesses appear on the marketplace.
- As an admin, I want to reject or suspend vendors so that unsafe or fraudulent vendors are blocked.
- As an admin, I want to manage customers so that platform misuse can be handled.
- As an admin, I want to view fraud signals so that suspicious behavior can be investigated.
- As an admin, I want analytics dashboards so that platform health and growth are visible.
- As an admin, I want to manage advertisements so that the platform can support vendor promotion.
- As an admin, I want to manage subscription plans so that premium vendor services can be introduced.
- As an admin, I want reports so that operations can make informed decisions.
- As an admin, I want security logs so that sensitive actions can be audited.
- As an admin, I want to moderate reviews and content so that the marketplace stays trustworthy.
- As an admin, I want role-based permissions so that team members only access the tools they need.

## Future Delivery Riders

- As a rider, I want to register and verify my identity so that I can deliver orders.
- As a rider, I want to receive delivery requests so that I can accept jobs.
- As a rider, I want to update delivery status so that customers and vendors know progress.
- As a rider, I want to view earnings so that I can track completed deliveries.

## Open-Source Contributors

- As a contributor, I want a clear README so that I understand the project quickly.
- As a contributor, I want contribution guidelines so that I know how to submit changes.
- As a contributor, I want issue templates so that bug reports and feature requests are useful.
- As a contributor, I want CI checks so that quality is maintained.
- As a contributor, I want API and developer documentation so that I can work safely on the platform.
