# AkaraConnect Phase 1 Planning Pack

AkaraConnect is a production-ready open-source marketplace platform by ZaxeeFix Enterprise for connecting customers with local Akara, Pap/Ogi/Akamu, and other local food vendors across Nigeria.

Phase 1 is planning only. No application code, dependency installation, database migrations, or deployment setup should be started until Phase 2 is approved.

## Core Vision

Build Africa's largest digital marketplace for local street food vendors.

## Mission

Digitize local food businesses and connect customers with trusted vendors using modern technology.

## Planned Tech Stack

- Frontend: Next.js, React, TypeScript, Tailwind CSS
- Backend: Node.js, Express.js, TypeScript
- Database: PostgreSQL on Neon
- ORM: Prisma
- Authentication: NextAuth, JWT, Google Login, Phone OTP
- Storage: Cloudinary
- Maps: Google Maps API, Google Places API, Geolocation API
- Payments: Paystack, Flutterwave, Moniepoint, Opay, PalmPay, Visa, Mastercard, Bank Transfer, Wallet, Cash on Delivery
- Frontend hosting: Vercel
- Backend hosting: Render
- Database hosting: Neon
- Version control: GitHub
