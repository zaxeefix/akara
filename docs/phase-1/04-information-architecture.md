## Navigation Principles

- Customers should reach vendor discovery, cart, orders, and favorites quickly.
- Vendors should see order status and menu tools immediately after login.
- Admins should land on operational dashboards with approval and risk queues.
- Language and theme controls should be accessible from user settings and public navigation.
- Mobile navigation should prioritize search, nearby vendors, orders, and account access.
