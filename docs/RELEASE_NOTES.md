### Known Limitations

- Production provider credentials are not configured in source.
- Live Google OAuth, Phone OTP, and 2FA providers require real credentials and final integration.
- Cloudinary upload endpoints need production provider testing.
- Paystack live payment flow requires real public and secret keys.
- Automated E2E tests are planned but not yet comprehensive.
- Git repository state must be repaired before commit/push.
