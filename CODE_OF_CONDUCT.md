# Code of Conduct

AkaraConnect contributors are expected to be respectful, constructive, and inclusive.

## Expected Behavior

- Use welcoming language.
- Respect different experience levels.
- Give actionable feedback.
- Focus criticism on ideas and code, not people.

## Unacceptable Behavior

- Harassment or discrimination.
- Personal attacks.
- Publishing private information.
- Sharing secrets, credentials, or sensitive user data.

Project maintainers may remove, edit, or reject contributions that violate this code of conduct.
